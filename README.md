# Pima Datacenter Firewall Project
Capstone project: designing and deploying a secure virtual network using a Physical Palo Alto Next-Generation Firewall in Pima Community Center's Datacenter

Overview
A capstone project designing and deploying a secure virtual 
network using a Palo Alto Next-Generation Firewall (NGFW). 
The goal was to simulate a small company environment with a 
firewall that could monitor traffic, enforce access controls, 
and support secure remote administration.

 Tools Used
- **Palo Alto NGFW** — Primary firewall and security platform
- **Proxmox** — Virtualization platform hosting all VMs
- **Ubuntu Desktop** — Internal client VMs for testing
- **GlobalProtect VPN** — Remote access and management
- **Palo Alto CLI + GUI** — Firewall configuration and monitoring

 Skills Demonstrated
- Enterprise firewall deployment and configuration
- Network zone segmentation (Trust, Untrust, DMZ)
- Routing protocol configuration and NAT policy design
- VPN setup and remote administration
- Troubleshooting routing and connectivity issues
- Technical documentation and report writing

 Key Achievements
- Deployed and configured Palo Alto NGFW in a virtualized 
  environment
- Created Trust, Untrust, and DMZ network zones with 
  enforced security policies
- Configured NAT rules enabling internal traffic to reach 
  the internet securely
- Enabled GlobalProtect VPN for remote firewall management
- Achieved full remote GUI and CLI access to the firewall
